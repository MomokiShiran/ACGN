## 快速开始

### 本地运行

1. 克隆项目
```bash
git clone https://github.com/MomokiShiran/ACGN.git
cd ACGN
```

2. 启动本地服务器
```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx http-server
```

3. 访问 `http://localhost:8000`

## 技术栈

- **HTML5** - 页面结构
- **CSS3** + **Bootstrap 4** - 样式框架
- **JavaScript** + **jQuery** - 交互逻辑
- **JSON** - 数据存储
